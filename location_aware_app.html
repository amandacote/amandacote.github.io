Location Aware App

Location Aware Alarm App that I created using AdobeXD

Live Demo

Project description

This was a project for my Information Design class and I had to design an app that was location aware. It had to have a feature that made it stand out from the competition. The new features that I added were facial recognition software which would be used to scan your face to turn off the alarm. This is helpful for heavy sleepers. I also added a questionaire that centered around sleeping habits, and depending on how they answered they would be offered to have the scanner as their feature to use.

Role

Design and creation of the app.

Challenge:

Figuring out how I was going to design the face scanner.
